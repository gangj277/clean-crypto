# Clean Crypto — 전략 방향서

## 문제 정의

한국 암호화폐 리딩방 시장은 구조적 신뢰 부재 상태다.

- 가상자산 투자자 **1,077만 명**, 하루 거래 규모 **6.4조 원**
- 2년간 리딩방 사기 피해액 **1조 2,901억 원**, 신고 **14,629건**
- 주요 수법: 선행매매, 펌프앤덤프, 수익률 조작, 다단계 구조
- 텔레그램/카카오톡 기반 운영 → 투명성과 책임 추적 불가능
- 법제도는 범죄의 진화 속도를 따라가지 못하는 상황

피해자는 20~50대 전 연령대에 걸쳐 있으며, 전 재산을 잃거나 극단적 선택에 이르는 사례까지 발생하고 있다.

## 핵심 인사이트

> 리딩방은 없어질 수 없다. 자기보다 투자를 잘하는 사람의 의견을 따르는 것은 인간의 본성이다.
> 본성을 이기려 하지 말고, **깨끗한 운영자만 신뢰받는 구조**를 만든다.

## 3-Phase 전략

### Phase 1 — 리딩방 클린 체크 (유저 획득)

**"당신이 받고 있는 리딩방, 정말 깨끗한가요?"**

리딩방을 이용하는 팔로워가 자기가 쓰는 리딩방의 정보를 입력하면, 어떤 면에서 클린하고 어떤 면에서 리스크가 있는지 분석해주는 앱.

- **타겟 유저**: 리딩방을 이용 중인 팔로워 (소비자향)
- **유저 심리**: "내가 쓰는 리딩방, 진짜 믿어도 되나?"라는 본질적 불안감
- **핵심 가치**: 리딩방의 클린함을 객관적으로 평가받을 수 있는 최초의 도구
- **목표**: MAU 확보 + 리딩방 시장의 신뢰 데이터 축적

분석 기준 예시:
- 수익률 보장 문구 여부 (레드플래그)
- 수수료 구조의 투명성
- 시그널의 구체성 (진입가/손절가/목표가 명시 여부)
- 운영 기간 및 일관성
- 다단계/리퍼럴 구조 여부
- 공개 트랙레코드 존재 여부

### Phase 2 — 클린 리딩방 프로모션 (수익화)

MAU가 잡히기 시작하면, 실제로 클린하게 운영되는 리딩방들에 컨택한다.

- **접근 방식**: "당신의 리딩방이 클린하다고 분석됐습니다. 파트너십을 맺고 프로모팅해드리고 싶습니다."
- **수익 모델**: 검증된 클린 리딩방의 광고 / 프로모션 수수료
- **리딩방 운영자 입장**: 클린 인증 = 차별화 수단 → 참여 인센티브 존재
- **팔로워 입장**: "이 앱에서 검증된 리딩방"이라는 신뢰 기반 탐색

이 단계에서 자생 가능한 광고 기반 플랫폼으로 성장한다.

### Phase 3 — 자체 리딩방 플랫폼 (엔드게임)

프로모팅을 받던 리딩방들이 우리 플랫폼 위에서 직접 리딩방을 운영하게 한다.

- 텔레그램 → 자체 플랫폼으로 이전
- **실시간 트래킹**: 모든 시그널, 매매 기록, 성과를 플랫폼이 직접 로깅
- **구조적 투명성**: 선행매매·수익률 조작·펌프앤덤프를 시스템 레벨에서 방지
- 이전 동기: Phase 1~2에서 축적된 브랜드 신뢰 + 플랫폼만의 고유 기능 제공
- 완전한 데이터 통제와 감사 추적이 가능한 신뢰 인프라 완성

## 왜 이 순서인가

Phase 1에서 소비자의 불안감으로 유저를 모으고, Phase 2에서 클린 공급자를 연결하고, Phase 3에서 양쪽을 자체 플랫폼에 가두는 구조다. 각 단계가 다음 단계의 전제 조건을 만든다.

| Phase | 핵심 자산 | 다음 단계로의 연결 |
|-------|-----------|-------------------|
| 1 | MAU + 신뢰 데이터 | 광고주(클린 리딩방)에게 유저 기반 제시 |
| 2 | 클린 리딩방 네트워크 + 광고 수익 | 검증된 리더 풀을 자체 플랫폼으로 초대 |
| 3 | 완전한 투명성 인프라 | 시장의 신뢰 표준이 됨 |
